# Project Rules

1. Please commit to Git every time after making changes to the project.  
   **Note:** Always write commit messages in English.

2. Please add a meaningful commit message every time you commit to Git.  
   **Note:** Commit messages must be in English.

3. Please create a new branch every time you make major changes to the project.  
   **Note:** Branch names and related documentation must be in English.

4. Please add a new paragraph to the Change_record.md file in the project root directory every time you make changes to the project.  
   **Title:** "Change Information"  
   **Content:** "Description of the change"  
   **Note:** All descriptions must be written in English.

5. Please add a new paragraph to the Change_record.md file in the project root directory every time you make major changes to the project.  
   **Title:** "Major Changes"  
   **Content:** "Description of the major changes"  
   **Note:** All descriptions must be written in English.

6. When you make changes to the project, you need to modify the README.md file in the project root directory.  
   **Note:** All README.md updates must be written in English.
