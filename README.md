# 2026 Ningbo Spring Festival Web Application

## Overview

The 2026 Ningbo Spring Festival Web Application is a comprehensive cultural platform designed to showcase the rich traditions and vibrant celebrations of the Spring Festival in Ningbo, Zhejiang Province, China. This interactive website offers users an immersive experience to explore Ningbo's 7,000-year Hemudu cultural heritage through various sections including knowledge, customs, food, games, and performances.The webpage is generated through artificial intelligence technology.

## Core Features

### 1. Cultural Showcase
- **Spring Festival Knowledge**: Comprehensive information about the history and significance of Spring Festival
- **Ningbo Customs**: Detailed exploration of local traditions and festive practices unique to Ningbo
- **Flower Market**: Virtual tour of Ningbo's famous Spring Festival flower markets
- **Food Culture**: Showcase of traditional Ningbo cuisine enjoyed during Spring Festival
- **Folk Performances**: Information about lion and dragon dances, and other traditional performances

### 2. Interactive Games
- **Riddle Guessing**: Challenge users with Ningbo-themed lantern riddles
- **Couplet Matching**: Educational game to learn about traditional Chinese couplets
- **Customs Quiz**: Test users' knowledge of Ningbo Spring Festival traditions
- **Leaderboard System**: Track and display user scores across all games

### 3. Visual Experience
- **Responsive Design**: Optimized for both desktop and mobile devices
- **Animated Elements**: Smooth animations and transitions throughout the website
- **Gradient Color Scheme**: Traditional Chinese red and gold color palette
- **Decorative Elements**: Festive icons and visual motifs representing Spring Festival

## How to Use

### 1. Local Development
1. Clone or download the project files
2. Open `index.html` in a modern web browser
3. Explore the website through navigation links
4. Try the interactive games by clicking on the game cards

### 2. Navigation Tips
- Use the navbar at the top to access different sections
- Scroll down the homepage to discover all content
- Click on "Back to Top" button to return to the top

### 3. Playing Games
1. Navigate to the Games page from the navbar
2. Select a game based on your interest and difficulty level
3. Read the game instructions carefully
4. Click on your answers or make selections
